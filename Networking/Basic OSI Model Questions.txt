Device Mapping Questions

Which OSI layer does a switch operate on?
data link layer


Which layer does a router operate on?
network layer


At which OSI layer does a firewall typically work?
network layer and transport layer


Where does a load balancer fit in the OSI model?
application layer and transport layer

Troubleshooting & Scenario Questions

If a server is reachable by IP but not by hostname, which OSI layer is likely having an issue?
application layer


If a cable is unplugged, which OSI layer is affected?
physical layer


If packets are reaching the destination but data is corrupted, which layer should be checked?
presentation layer

At which OSI layer does SSL/TLS operate conceptually?
presentation layer
