what is the difference between git and GitHub.


Git → Version control tool installed locally to track code changes.

GitHub → Cloud-based platform to host repositories, collaborate via PRs, issues, actions, etc.

2. What is a repository?
A place where project files + history are stored.

Local repo → On your machine (.git folder)

Remote repo → Hosted on GitHub/GitLab/Bitbucket
