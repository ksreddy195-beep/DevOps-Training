Python Variables
1. What is a Variable?

A variable is a name given to a value stored in memory.

In Python, variables are used to store data that can be used and changed later in a program.

Example:

x = 10
name = "Python"


Here:

x stores the value 10

name stores the value "Python"

2. Why Do We Need Variables?

Variables help to:

Store data

Reuse values

Make programs readable

Perform operations on stored data

Example without variable:

print(10 + 10)


Example with variable:

a = 10
print(a + a)

3. Creating Variables in Python

Python does not require any special keyword to declare a variable.

Syntax:

variable_name = value


Examples:

age = 25
price = 99.5
city = "Hyderabad"
is_active = True

4. Rules for Naming Variables

Variable name must start with a letter (a-z, A-Z) or underscore (_)

It cannot start with a number

It can contain letters, numbers, and underscores

Variable names are case-sensitive

Keywords cannot be used as variable names

Valid names:

name
user_age
_total
price1


Invalid names:

1name
user-age
class
total price

5. Case Sensitivity in Variables

Python treats uppercase and lowercase letters as different.

Example:

age = 20
Age = 30


Here, age and Age are two different variables.

6. Assigning Multiple Variables
Assigning multiple values at once:
a, b, c = 10, 20, 30

Assigning same value to multiple variables:
x = y = z = 100

7. Variable Reassignment

Variables can be updated after creation.

Example:

count = 5
count = 10


The value of count becomes 10.

8. Dynamic Typing in Python

Python is dynamically typed, meaning the type of a variable can change.

Example:

x = 10
x = "Hello"
x = 5.5


Python automatically changes the data type.

9. Checking Variable Type

Use type() to check the data type.

Example:

x = 10
print(type(x))


Output:

<class 'int'>

10. Common Data Types Used with Variables
Data Type	Example
int	10
float	10.5
str	"Python"
bool	True, False
11. Deleting a Variable

Use del to delete a variable.

Example:

x = 100
del x



Accessing x after deletion will cause an error.

12. Best Practices for Variables

Use meaningful variable names

Follow snake_case naming convention

Avoid single-letter names except in loops

Keep variable names short and clear

Example:

total_marks = 450
student_name = "Ravi"

13. Common Mistakes

Using reserved keywords as variable names

Forgetting case sensitivity

Using unclear variable names

Using spaces in variable names
