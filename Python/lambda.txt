Definition
A lambda function is a small, anonymous function (a function without a name).


It can take any number of arguments


It can have only one expression


The expression is evaluated and returned automatically



Syntax
lambda arguments : expression


Basic Examples
Example 1: Add 10 to a number
x = lambda a: a + 10
print(x(5))

Output
15


Example 2: Multiply two numbers
x = lambda a, b: a * b
print(x(5, 6))

Output
30


Example 3: Add three numbers
x = lambda a, b, c: a + b + c
print(x(5, 6, 2))

Output
13


Why Use Lambda Functions?
Lambda functions are useful when:


You need a short, simple function


The function is used only once


You want to pass a function as an argument to another function


They make code more concise and readable for small operations.
--------------------------------------------------------------------------------------------

Lambda Inside Another Function
Example: Function returning a lambda
def myfunc(n):
    return lambda a: a * n


Example 1: Double a number
mydoubler = myfunc(2)
print(mydoubler(11))

Output
22


Example 2: Triple a number
mytripler = myfunc(3)
print(mytripler(11))


Output
33


Example 3: Using both in the same program
mydoubler = myfunc(2)
mytripler = myfunc(3)

print(mydoubler(11))
print(mytripler(11))

Output
22
33

Lambda with Built-in Functions
Lambda functions are commonly used with:


map()


filter()


sorted()



Lambda with map()
Purpose
Applies a function to each item in an iterable.
Example: Double all numbers in a list
numbers = [1, 2, 3, 4, 5]
doubled = list(map(lambda x: x * 2, numbers))
print(doubled)

Output
[2, 4, 6, 8, 10]


Lambda with filter()
Purpose
Filters elements based on a condition (function returns True).
Example: Get odd numbers from a list
numbers = [1, 2, 3, 4, 5, 6, 7, 8]
odd_numbers = list(filter(lambda x: x % 2 != 0, numbers))
print(odd_numbers)

Output
[1, 3, 5, 7]


Lambda with sorted()
Purpose
Sorts elements using a custom key.

Example 1: Sort tuples by second value
students = [("Emil", 25), ("Tobias", 22), ("Linus", 28)]
sorted_students = sorted(students, key=lambda x: x[1])
print(sorted_students)

Output
[('Tobias', 22), ('Emil', 25), ('Linus', 28)]


Example 2: Sort strings by length
words = ["apple", "pie", "banana", "cherry"]
sorted_words = sorted(words, key=lambda x: len(x))
print(sorted_words)

Output
['pie', 'apple', 'banana', 'cherry']


Key Points Summary


Lambda functions are anonymous, one-line functions


Syntax is short and concise


Best suited for temporary or simple logic


Commonly used with map(), filter(), and sorted()


Not suitable for complex logic or multiple expressions
