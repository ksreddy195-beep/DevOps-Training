Task 14:
Read a file line by line and print each line with line number

Example:
#!/bin/sh

file="input.txt"
line_no=1

while IFS= read -r line
do
    echo "$line_no: $line"
    line_no=$((line_no + 1))
done < "$file"

Here IFS is Internal Field Separator

Task 17:
Create a function that:

Accepts a filename

Prints file size and permissions

file_info() {
    file="$1"

    if [ ! -e "$file" ]; then
        echo "File does not exist"
        return 1
    fi

    size=$(stat -c %s "$file" 2>/dev/null || stat -f %z "$file")
    perms=$(stat -c %A "$file" 2>/dev/null || stat -f %Sp "$file")

    echo "File: $file"
    echo "Size: $size bytes"
    echo "Permissions: $perms"
}

---
other approach to above one:

file_info() {
    file="$1"

    if [ ! -e "$file" ]; then
        echo "File does not exist"
        return 1
    fi

    size=$(ls -lh "$file" | awk '{print $5}')
    perms=$(ls -l "$file" | awk '{print $1}')

    echo "File: $file"
    echo "Size: $size"
    echo "Permissions: $perms"
}


awk '{print $5}'

awk splits the line into fields (columns) using spaces

Field numbers:

$1 → permissions
$2 → hard links
$3 → owner
$4 → group
$5 → size
$6 → month
$7 → day
$8 → time/year
$9 → filename
